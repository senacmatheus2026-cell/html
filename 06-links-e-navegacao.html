Link para site externo

Link Externo aponta para um endereço (URL) fora do seu site ou servidor onde seus arquivos estão hospedados

Como ele funciona

Para que o navegador saiba que deve sair do seu domínio, o atributo href do elemento <a > eve conter o endereço completo so site externo, incluindo o protocolo (http:// ou https://).

Atributo target="_blank

O atributo target="_blank e usado para abrir o link em uma nova aba ou janela do navegador, permite que o usário mantenha a página atual aberta enquanto visita o site externo.
